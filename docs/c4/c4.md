# C4 Context Diagram — Logo Visualizer

Paste into **plantuml.com/plantuml**

```plantuml
@startuml

!include https://raw.githubusercontent.com/plantuml-stdlib/C4-PlantUML/master/C4_Context.puml

LAYOUT_WITH_LEGEND()

title C4 Context Diagram - Logo Visualizer

Person(admin, "Admin", "Ops aetter produkter og definerer\nprintzoner i admin-modulet")
Person(customer, "Slutkunde", "Uploader logo og forhaandsviser\ndet paa et produkt")
Person(salesperson, "Saelger", "Forhaandsviser logo paa produkt\nvia delt URL og downloader mockup")

System(logovis, "Logo Visualizer", "Lader brugere uploade et logo og\nforhaandsvise det i en printzone paa et produkt.\nBestaar af admin-app, viewer-app og backend-API.")

System_Ext(master, "Master Application", "Udsteder JWT-tokens til admin-brugere.\nEkstern autentifikationsloesning.")
System_Ext(midocean, "Midocean CDN", "Leverer produktbilleder til preview\nog eksport-compositing.")
SystemDb_Ext(db, "SQL Server", "Gemmer produkter, printzoner\nog tillladte teknikker.")

Rel(admin, logovis, "Ops aetter produkter og zoner", "HTTPS / REST API")
Rel(customer, logovis, "Uploader logo og downloader PNG", "HTTPS / REST API")
Rel(salesperson, logovis, "Forhaandsviser og deler mockup via URL", "HTTPS / REST API")

Rel(logovis, master, "Validerer JWT-token", "HTTPS")
Rel(logovis, midocean, "Henter produktbilleder", "HTTPS")
Rel(logovis, db, "Laaser og skriver produkt-data", "EF Core / SQL")

@enduml
```
